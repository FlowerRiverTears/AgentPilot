# 第二版开发路线

## 目标

第二版目标是把第一版可演示工作台升级为可长期使用的智能体管理平台。

核心方向：

- 智能体可编辑、可删除、可复制、可发布。
- 前台只展示已发布智能体。
- 工具系统从内置示例升级为可配置真实工具。
- 后台有基础登录和权限。
- 运行历史可查询、可追踪、可定位失败原因。
- 前台支持流式输出，复杂回答体验更自然。

## 第二版验收流程

1. 管理员登录后台。
2. 创建并发布智能体。
3. 绑定知识库和真实 HTTP 工具。
4. 前台只看到已发布智能体。
5. 用户提问。
6. 系统按需检索知识库或调用工具。
7. 前台流式显示回答。
8. 后台查看运行历史和详情。

## 不进入第二版范围

- 多租户计费。
- 复杂审批流。
- Agent Studio 工作流画布。
- 大规模分布式调度。
- 自动优化和评测闭环。
