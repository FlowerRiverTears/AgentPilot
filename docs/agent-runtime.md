# 智能体运行时

## 当前运行流程

入口：

```text
POST /api/runs
```

流程：

1. 根据 `agent_id` 读取智能体配置。
2. 根据用户输入检索智能体绑定的知识库。
3. 根据用户输入和工具绑定情况调用应用工具。
4. 拼接系统提示词、用户问题、知识库上下文、工具结果。
5. 调用模型网关生成回答。
6. 保存运行记录和执行步骤。
7. 返回回答、引用来源、执行过程和模型信息。

## 上下文结构

后端会把上下文组织成：

```text
用户问题：
...

知识库上下文：
...

应用工具结果：
...

请给出清晰回答。优先使用应用工具结果回答实时业务数据；
使用知识库内容时说明来源；如果没有可靠依据，请明确说明无法确认。
```

## 执行步骤

当前返回的步骤包括：

- 接收任务
- 知识库检索
- 应用工具调用
- 大模型生成

## 错误处理

当真实大模型调用失败时，后端会返回错误说明，提示用户检查：

- 模型接口地址
- API Token
- 模型名
- 后端服务

## 第二版增强

- Run 列表和 Run 详情页已完成。
- 引用来源和执行步骤已可在运行历史中回看。
- 流式输出。

- Run 详情页展示每一步耗时。
- 工具调用结果单独入库。
- 支持失败重试。
