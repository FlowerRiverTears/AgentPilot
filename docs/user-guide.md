# 用户使用教程

## 访问地址

- 后台工作台：http://localhost:5173
- 前台体验：http://localhost:5173/portal
- 使用教程：http://localhost:5173/guide

## 第一步：配置模型

进入后台的“模型配置”页面。

填写：

- 配置名称
- Base URL
- API Key
- 默认模型名

填写完成后点击“测试”。测试通过后再创建智能体。

## 第二步：创建知识库

进入“知识库”页面。

操作：

1. 创建知识库。
2. 选择知识库。
3. 上传文档。
4. 输入问题做检索测试。

知识库适合放稳定资料，例如产品手册、FAQ、制度文档。

## 第三步：创建智能体

进入“智能体后台”页面。

填写：

- 名称
- 描述
- 模型配置
- 系统提示词
- 知识库
- 应用工具

创建后，智能体会出现在前台。

## 第四步：前台体验

点击后台右上角“进入前台”，或访问：

```text
http://localhost:5173/portal
```

前台只展示用户对话体验，不显示后台管理菜单。

## 第五步：查看运行过程

后台“运行测试”页面可以选择智能体并执行任务。

运行后会看到：

- 回答
- 执行过程
- 引用来源

也可以进入“运行历史”页面，查看最近的运行记录和单次运行详情。

## 回答格式

系统会自动处理：

- `<think>` 思考过程折叠显示。
- Markdown 代码块显示为灰色代码框。
- 清理标题符号和分隔线。

## 主题切换

前台和后台右上角都有深色 / 亮色切换按钮。

主题会保存在浏览器中。

## 常见问题

### 模型测试 401

通常是 API Key 无效、过期或填错。

### 回答慢或复杂问题失败

后端和前端超时时间已经调长，但复杂模型仍可能需要等待。可以检查后端日志和模型服务状态。

### 前台没有智能体

先到后台创建智能体。第二版会增加发布状态，前台只展示已发布智能体。
